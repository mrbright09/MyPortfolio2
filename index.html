Flagship Project

Enterprise AI Operations Agent

Claude AI · ServiceNow · New Relic · Microsoft 365 · SharePoint · TargetProcess · GitHub · Teams · Outlook

Built a production AI agent that serves as a single command interface for enterprise IT operations. The agent autonomously correlates signals across 8 connected systems — pulling ServiceNow incidents, New Relic telemetry, SharePoint runbooks, and project records simultaneously — then surfaces a direct answer instead of raw system output. It can post Teams messages, send emails, update Excel trackers, create SharePoint items, and maintain OneNote records without switching platforms.

8 enterprise systems unified in one interface Multi-system triage collapsed into a single workflow Eliminates manual search across ServiceNow, New Relic & SharePoint Drafts & sends Teams/email/SharePoint updates autonomously Faster triage — related incidents, alerts & KBs in one response

Safe Senders Automation & Intune Enforcement

Power Automate · Azure Automation · Exchange Online · SharePoint · Intune · Microsoft Graph API

Built a Power Platform–driven Safe Senders management solution that replaced a manual, network-share-dependent process with a fully automated, auditable multi-tenant workflow. SharePoint acts as the master source of truth. Power Automate updates the file and fires a webhook to Azure Automation, which connects to Exchange Online via app-based auth, calculates per-tenant add/remove changes, applies them, and logs every run to a SharePoint list. Intune remediation scripts push the safe senders file to Classic Outlook cached-mode endpoints — covering all three Outlook client types through the correct control path for each.

Replaced manual network-share process entirely Covers New Outlook, Classic online & Classic cached mode Multi-tenant safe sender management from one source of truth Full audit log written to SharePoint on every run App-based auth — no stored credentials

SQL Database Copy & Backup Automation

Azure Automation Runbooks · Azure Key Vault · Email Notifications

Built 7 Runbooks to fully automate SQL database copy/backup across all environments (Prod → Pre-Prod → QA → DEV). Integrated Key Vault for secure credential storage, auto-timestamped copies, Hyperscale tier optimization to cut storage costs, and automated email status logs on every run.

100% manual DB ops eliminated Hours of engineer time saved per sprint Zero credential exposure via Key Vault

Azure VM Temporary Size Upgrade Workflow

Power Automate · Outlook · Azure Automation Runbooks

Designed an end-to-end request/approval flow for temporary VM size upgrades. Parses inbound email requests, routes for manager approval, triggers the Runbook to resize, runs the job, then auto-reverts the VM to original size — with a full status report delivered at each step.

Auto-reverts — no human follow-up needed Approval gate prevents unauthorized scaling Eliminates idle oversized VM costs

Power App for SQL DB Copy/Delete

Power Apps · Power Automate · Azure Runbooks

Built a secure, non-technical interface that lets IT staff self-service database copy or deletion tasks without touching Azure Portal or writing a single line of code. Pulls live database lists, enforces timestamped naming conventions, and uses role-based access to ensure only authorized users can trigger operations.

Non-engineers can operate complex DB tasks 70%+ reduction in engineer ticket load Consistent naming & audit trail on every run

Microsoft Copilot Incident Review Agent

Microsoft Copilot Studio · Power Automate · ServiceNow · Microsoft Teams · JSON Orchestration

Developed an internal Copilot Agent that automatically analyzes ServiceNow incidents and generates structured operational summaries for leadership. Incident data flows from ServiceNow into Power Automate, gets passed to a Copilot Agent prompt for AI-driven pattern analysis, and returns a structured JSON response delivered directly to Microsoft Teams.

AI-driven root cause analysis from raw incident data Structured JSON output with confidence scoring Auto-delivered summaries to Microsoft Teams Eliminates manual weekend incident reporting
